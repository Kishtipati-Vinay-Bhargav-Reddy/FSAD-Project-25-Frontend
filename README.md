# 🎨 Online Assignment Submission System – Frontend

## 📌 Project Overview

This is the frontend of the **Online Assignment Submission and Grading System** developed using React.js.
It provides a user-friendly interface for students and faculty to interact with the system.

---

## 🚀 Features

* 🔐 User Authentication (Login/Register)
* 📊 Dashboard View
* 📚 View Assignments
* 📤 Submit Assignments
* ⚡ Real-time Form Validation
* 🎨 Responsive UI with CSS

---

## 🛠️ Tech Stack

* React.js
* Axios
* CSS / Bootstrap
* React Router

---

## 📂 Project Structure

src/
├── components/
├── pages/
├── services/
├── App.js
├── index.js

---

## 🔗 API Integration

The frontend communicates with the backend using REST APIs.

Example:
GET /api/assignments
POST /api/login

---

## ⚙️ Installation & Setup

```bash
git clone https://github.com/your-username/frontend-repo.git
cd frontend-repo
npm install
npm start
```

---

## 🌐 Backend URL

Make sure backend is running on:
http://localhost:8080
