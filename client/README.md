# Online Assignment Submission & Grading System

## Frontend
- React (Vite)
- Tailwind CSS
- Framer Motion
- React Router
- Axios
- Glassmorphism UI

## Backend
- Node.js + Express
- MongoDB (Mongoose)
- JWT Authentication
- Multer file upload

## Features
- Role-based access (student/teacher)
- Assignment management
- Submission system
- Dashboards
- Secure authentication

## Setup

### Backend
1. Install dependencies: `npm install`
2. Create `.env` file (see sample)
3. Start server: `npm start`

### Frontend
1. Install dependencies: `npm install`
2. Start client: `npm run dev`

## Folder Structure
- server/
- client/

## Production
- Build frontend: `npm run build`
- Serve static files or deploy separately

## Security
- Password hashing
- JWT secret
- Role middleware
- Input validation
- Error handling

## File Upload
- Files stored in `/uploads` (backend)
- Accessible via API

## Environment
- MongoDB required
- Node.js required
